# Project Overview

Asthma remains one of the most common chronic respiratory diseases worldwide, affecting millions of people across all age groups.

This project leverages the Global Burden of Disease (GBD) dataset to analyze:

* Global asthma prevalence
* Mortality rates
* Age-specific burden
* Gender-based differences
* Geographic distribution
* Historical trends from 1990–2023

The project transforms complex healthcare data into actionable insights through interactive dashboards and AI-powered exploration.

# AI Chatbot

## AsthmAI – Intelligent Healthcare Assistant

AsthmAI is an AI-powered chatbot integrated into the platform to enhance user interaction with asthma-related data.

### Features

* Answer questions about asthma
* Explain medical indicators and metrics
* Analyze uploaded GBD datasets
* Generate data-driven insights
* Create visual summaries and charts
* Support healthcare research exploration
* Provide prevention and awareness guidance

### Example Questions

* Which countries have the highest asthma prevalence?
* How has asthma mortality changed since 1990?
* Which age groups are most affected?
* Compare asthma burden between males and females.
* Show asthma trends for a specific country.

# Problem Statement

Asthma continues to impose a significant health burden worldwide. Despite the availability of public health data, extracting meaningful insights from large-scale healthcare datasets remains challenging.

This project aims to bridge that gap by providing a centralized analytical platform that helps researchers, students, and decision-makers understand asthma patterns and trends.
